# Movie-App
<image src="https://github.com/Daniel0110000/Movie-App/blob/master/app/src/main/res/mipmap-xxhdpi/ic_launcher.png"/>

<h3>Application functionalities:</h3>
<ul>
  <li>Show all movise</li>
  <li>Search for movie</li>
  <li>Show your favorite movies</li>
  <li>Add movie to favorites</li>
  <li>Delete movie from favorites</li>
  <li>Show movie details</li>
</ul>

<h3>Resources used:</h3>
<ul>
  <li>Retrofit</li>
  <li>Coroutines</li>
  <li>Picasso</li>
  <li>Dagger Hilt</li>
  <li>Gson</li>
  <li>ViewModel</li>
  <li>LiveData</li>
  <li>Api: <a>https://www.episodate.com/api</a></li>
</ul>
